# auto-gacha

Automating gacha dailies for certain games.

Program
- Sikulix (python, java)

### Main Features:
- Select which window to focus on for each respective game
- Check boxes to select which game to automate

5. Perform Equipment or Fairy Calibration once.

8. Give a gift to a T-Doll or Coalition Unit.
9. Defeat Boss--
10. Perform 3 Resource Recoveries.
11. Perform an exploration at the Foward Basecamp.
12. Perform a Capture Operation (either a Standard Capture or Aided Capture).
13. Perform 2 Doll Productions.
14. Equipment Production--

    Features:

    - Say what was sucessfully done, what failed, what was skipped
